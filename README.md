# The Impossible Triangle — 不可能三角

[![Live Demo](https://img.shields.io/badge/Live-impossible--triangle.vercel.app-e04040)](https://impossible-triangle.vercel.app)

Interactive bilingual guide to the Impossible Triangle — why you can never optimize three things at once. Featuring an interactive simulator and 20 real-world examples.

## Features

- **Interactive Triangle Simulator** — drag sliders to see trade-offs in real time (budget-constrained)
- **20 Examples** across software, blockchain, AI, economics, databases, automotive, tires, healthcare, semiconductors, food, platforms, startups, energy, cybersecurity, education, government, photography, life, college, and housing
- **Why They Exist** — resource scarcity, Pareto optimality, thermodynamics, complexity theory
- **Can You Break It?** — technology leaps, new materials, better algorithms, time shifting, reframing
